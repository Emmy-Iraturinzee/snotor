Fonsen Logistics

SaaS Platform for Asian logistic company

About

Industry: Logistics
Location: Singapore
Type of software: SAAS Platform
Scope of work: Back-end, Front-end, UI/UX Design
Fonsen Logistics is a company that provides services on warehousing, packing, delivery, and others. The client’s main goal was to make the delivery process for different restaurants and their goods suppliers more efficient.
So they needed to create a platform where restaurant managers can see all the information about their orders and track them.

Technologies

Ruby

Rails

React.js

Postgres DB

Redis

TypeScript

Yup

Formik

Lodash

Axios

Key features

  • Ability to track the orders and see their latest status and other important details in online mode (finished/ returned/ changed address/ average time spend on the delivery etc.)
  • Ability to create different user roles in the company’s profile and send invitations to join the system via email
  • In-app possibility for conversations about selected orders for solving delivery issues
  • Ability to generate PDF file with QR code, barcode, and information about the purchase, which can be distributed to administrators in order to stick them on the orders. Users are able to scan the codes and see where their orders are located
  • Ability to add the Excel file with the info on the remaining inventory to the order page
