Inokufu Search

Education Search Chrome Extension

About

Industry: Education
Location: France
Type of software: Chrome Extension
Scope of work: Project Management, Business Analysis, Front-end, UI/UX Design, Quality Assurance
This Chrome extension helps users find the best learning objects on a specific topic. Learning Objects (LO) can be educational youtube videos, apps, books/ebooks, podcasts, MOOCs, websites, etc.
Inokufu has built an extensive library of LO that can be accessed through an API.

Technologies

Node.js

React.js

React Context

Challenge

To create a useful, functional interface with a great speed for helping people, who would like to learn new knowledge from the full internet space in one place.

Requirements

  • It was necessary to develop a useful Chrome extension for French and English speaking users.
  • Implement the ability to automatically open and use the extension when a user searches for something in the browser.
  • Develop the specific unique view of extension for ten search engines when it’s embedded in web pages.
  • Implement light and dark modes for different users depending on their browser themes.
  • Develop the ability to become a premium user to have more opportunities to learn.
  • It is necessary to create the unique categories of learning objects as a tag slider with the ability to freeze it.
  • Implement the ability to send the feedback about the extension fast and easy in two clicks.

Results

We created a Chrome extension that allows users to access the necessary courses using a advanced, intuitive system with tabs and filters, a high-quality design and a well-thought-out customer experience. The system collects feedback about itself, which allows Inokufu to improve it, raise CSI and NPS, and increase the value for users as well.
